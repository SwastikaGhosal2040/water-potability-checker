# Water Potability Checker

This project is a web application that checks whether water is safe to drink based on water quality parameters.

### Live Application
https://swastikaghosal2040-water-potability-checker-app-fzhuvo.streamlit.app/

### Parameters Used
- pH
- Arsenic
- Lead
- Fluoride
- Nitrate
- Iron

The system compares input values with safe drinking water standards and determines whether the water is drinkable.
